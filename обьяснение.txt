папка img - картинки
папка pages - страницы
папка styles - стили которые имеют такое же название как и страницы
файл style.css с универсальными стилями которые используются для всех (или почти) страниц
главная страница index.html в корне папки - название такое потому что стандарт
подчеркивание 


вот это 
.ref:after {
	display: block;
	position: absolute;
	left: 0; /*изменить на right:0;, чтобы изменить направление подчёркивания */
	width: 0;/*задаём длинну линии до наведения курсора*/
	height: 2px; /*задаём ширину линии*/
	background-color: #6b410a; /*задаём цвет линии*/
	content: "";
	transition: width 0.3s ease-out; /*задаём время анимации*/
}
.ref.b:after {
  background-color: black;
}
.ref:hover:after,
.ref:focus:after {
	width: 100%; /*устанавливаем значение 100% чтобы ссылка подчёркивалась полностью*/
}

для расположения дочерних элементов например в main тегах использовал display: flex;

для анимаций при наведения курсора 
.product {
    transition: .2s; /* скорость анимации */
}
.product:hover { /* при наведении (главное слово - hover если спросят что это скажешь псевдокласс) */
    top: -2px; /* сверху передвегаем на -2 пикселя тобеж вверх */
    box-shadow: 0 2px 3px rgba(0, 0, 0, 0.5); /* добавляем тень черного цвета с полупрозрачностью (она не прям черная) */
}
